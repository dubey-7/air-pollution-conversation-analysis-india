# 🌏 Air Pollution Conversations in India – Data Analysis

📊 This project analyzes **19K+ online conversations and media articles about air pollution in India** to identify trends, key themes, and platform engagement using **Excel and Power BI**.

---

## 🔎 Short Description
Interactive **Power BI dashboard** analyzing air pollution discussions in India, highlighting conversation trends, themes, and audience reach.

---

## 🎯 Project Objectives

✔ Identify and tag **irrelevant content** in the dataset  
✔ Analyze **monthly trends** in air pollution conversations  
✔ Discover **key discussion themes**  
✔ Visualize insights through a **Power BI dashboard**

---

## 📊 Dashboard Insights

📈 **Monthly Conversation Trends**  
📊 **Theme Distribution**  
🌐 **Platform-wise Conversation Volume**  
📢 **Audience Reach by Platform**  
✅ **Relevant vs Irrelevant Content**

Key observations:
- Conversation spikes in certain months indicating **increased public concern**
- **Twitter dominates** air pollution discussions
- Major themes include **environmental concerns, public awareness, and policy discussions**

---

## 🛠 Tools Used

- 📊 Excel – Data cleaning  
- 📈 Power BI – Dashboard visualization  
- 🔎 Data Analysis – Trend and thematic analysis
